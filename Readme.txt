# Проект с формой заявки на получение кредита

## Описание

Этот проект включает веб-страницу с таблицей, содержащей информацию о банковских кредитах. Пользователи могут подать заявку на кредит, заполнив форму в модальном окне. Данные заявки сохраняются в файл на сервере и подтверждаются пользователю.

## Структура проекта

- `index.html` — Основной HTML-файл для отображения страницы с таблицей и модальным окном.
- `styles.css` — CSS файл для стилизации страницы и модального окна.
- `script.js` — JavaScript файл для обработки событий и отправки данных формы.
- `apply.php` — PHP скрипт для обработки POST-запросов и сохранения данных в файл.

## Инструкции по запуску

### Требования

- PHP сервер (Я использовал XAMPP)
- Браузер для тестирования

### Шаги для запуска

1. **Установка веб-сервера:**
   - Скачайте и установите XAMPP (или другой сервер, поддерживающий PHP) с [официального сайта XAMPP](https://www.apachefriends.org/index.html).
   - Запустите XAMPP Control Panel и включите модули Apache и MySQL.

2. **Размещение файлов проекта:**
   - Переместите папку проекта в директорию `htdocs` в установке XAMPP. Обычно это `C:\xampp\htdocs\project_folder`.

3. **Запуск сервера:**
   - Убедитесь, что сервер Apache работает. В XAMPP Control Panel нажмите "Start" для модуля Apache.

4. **Открытие проекта:**
   - Откройте браузер и перейдите по адресу `http://localhost/project_folder/index.html`, заменив `project_folder` на имя папки.

5. **Тестирование формы:**
   - Заполните форму и отправьте заявку. Данные будут сохранены в файл `applications.txt`, который находится в той же директории, что и `apply.php`, если файла `applications.txt` нет то он будет создан автоматически после отправки формы.

## Проблемы и устранение неполадок

- **Ошибка `405 Method Not Allowed`:** Убедитесь, что сервер правильно настроен и файл `apply.php` доступен по указанному пути.
- **Ошибка `Network response was not ok`:** Проверьте конфигурацию сервера и наличие ошибок в логах.
